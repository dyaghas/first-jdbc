This is a simple project I created to learn how to connect a database to 
a Java application. It consists of a 'Car' and a 'Owner' class that
translates to two tables in a database.

Esse é um projeto que foi criado com a intenção de aprender como conectar 
um banco de dados a uma aplicação Java. O programa contém as classes 
'Car' e 'Owner' e duas tabelas para guardar as informações dessas tabelas.
